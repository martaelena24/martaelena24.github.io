¿Qué es GitHub?

Introducción

GitHub es una plataforma de desarrollo colaborativo que permite a los desarrolladores alojar y gestionar proyectos de software utilizando Git, un sistema de control de versiones distribuido. GitHub facilita la colaboración entre equipos y desarrolladores de todo el mundo.

El logo de GitHub, que representa a su icónica mascota Octocat, es reconocido mundialmente.

Características de GitHub

  • Repositorio de código: Almacena y gestiona proyectos de software.
  • Control de versiones: Permite rastrear cambios en el código a lo largo del tiempo.
  • Colaboración: Soporta trabajo en equipo a través de pull requests y revisiones de código.
  • Integraciones: Se puede integrar con herramientas de CI/CD, como GitHub Actions.
  • Documentación: Permite crear wikis y documentación para proyectos.

¿Cómo funciona?

GitHub utiliza Git para gestionar versiones de código. Los desarrolladores pueden crear "repositorios" para almacenar sus proyectos, hacer cambios en el código, y "commit" esos cambios para que otros puedan ver el historial de modificaciones. Los equipos pueden trabajar en diferentes ramas del proyecto al mismo tiempo y luego fusionar sus cambios.

Ejemplo de Código

A continuación, un ejemplo de cómo clonar un repositorio desde GitHub usando la terminal:

git clone https://github.com/usuario/repositorio.git

Este comando crea una copia local del repositorio especificado.
